agent-ready-extract-api

Extract clean content and metadata from public URLs as JSON.

Swagger UI OpenAPI Health

Use cases

Monitoring

Track changes on public pages by hashing extracted text.

Indexing

Convert messy HTML into clean text + headings for search or RAG.

Research

Quickly pull title/description/links from documentation and blogs.

Example request

POST /api/v1/extract
Content-Type: application/json

{
  "url": "https://example.com",
  "options": { "include_text": true, "include_title": true, "include_links": true }
}

Example response (shape)

{
  "url": "https://example.com",
  "fetch": { "fetched_at": "...", "final_url": "...", "status_code": 200, "content_type": "text/html; charset=utf-8" },
  "result": { "title": "...", "description": "...", "headings": ["..."], "text": "...", "links": [{"href":"...","text":"..."}] }
}
